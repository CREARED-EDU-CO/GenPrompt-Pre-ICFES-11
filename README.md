# Generador de Prompts para el ICFES SABER 11

Este repositorio contiene una aplicación web diseñada para ayudar a estudiantes de grado 11° en Colombia a prepararse de manera efectiva para el examen de estado **SABER 11**. La herramienta genera prompts (indicaciones) personalizados y optimizados para ser utilizados en modelos de lenguaje (como ChatGPT), permitiendo profundizar en temas específicos de las pruebas ICFES.

## 🎯 Objetivos
- **Reforzar conocimientos**: Facilitar el estudio de temas y subtemas evaluados en las diferentes áreas del examen.
- **Optimizar el aprendizaje**: Proveer estructuras de estudio especializadas (glosarios, resúmenes, ejercicios) mediante el uso de Inteligencia Artificial.
- **Accesibilidad**: Ofrecer una herramienta gratuita y fácil de usar para cualquier estudiante con acceso a internet.

## ✨ Características Principales

### 📚 Áreas de Conocimiento (Materias)
La aplicación cubre todas las áreas fundamentales evaluadas por el ICFES:
- **Lectura Crítica**
- **Matemáticas**
- **Ciencias Naturales**: Física, Química, Biología.
- **Ciencias Sociales**: Geografía, Historia, Competencias Ciudadanas.
- **Inglés**

### 🛠️ Funcionalidades del Generador
- **Selección Dinámica**: Permite elegir materia, tema principal y subtema específico.
- **Niveles de Competencia**: Adaptación del contenido según el nivel de habilidad deseado:
  - **Básico**: Enfocado en contenidos locales y conceptos fundamentales.
  - **Intermedio**: Relación entre conceptos y análisis básico.
  - **Avanzado**: Pensamiento crítico, análisis sistémico y resolución de problemas complejos.
- **Tipos de Actividades**: Generación de diversos formatos de estudio:
  - Glosarios y resúmenes detallados.
  - Tablas comparativas y analogías.
  - Mapas conceptuales y ejemplos prácticos.
  - Juegos interactivos y actividades lúdicas.
  - Preguntas de opción múltiple (con o sin respuestas explicadas).
  - Rúbricas de autoevaluación.

### 💻 Interfaz y Tecnología
- **Web Responsiva**: Diseño optimizado para dispositivos móviles y escritorio.
- **Integración con IA**: Botón directo para abrir el prompt generado en ChatGPT.
- **Copiado Rápido**: Función para copiar el prompt al portapapeles con un solo clic.
- **Sin Dependencias**: Funciona completamente en el navegador (HTML/CSS/JS puro).

## 🚀 Cómo Usar
1. Abre el archivo `index.html` en tu navegador.
2. Selecciona la **Materia**, **Tema** y **Subtema** que deseas estudiar.
3. Elige tu **Nivel de Competencia** y el **Tipo de Actividad** que prefieras.
4. Haz clic en **GENERAR PROMPT**.
5. Copia el resultado o ábrelo directamente en tu chatbot de IA preferido para comenzar tu sesión de estudio.
